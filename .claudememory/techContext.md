# Tech Context

- Frontend: React 19, TypeScript, Vite, React Router v7.
- State: Zustand with `persist` middleware.
- Backend: Firebase Auth and Firestore.
- i18n: `i18next` and `react-i18next`.
- Styling: Tailwind CSS v4 with Lucide React icons.
- Export tooling: `docx` and `file-saver`.
- Important scripts: `npm run dev`, `npm run build`, `npm run lint`.
- Firebase environment variables are read from `import.meta.env`: `VITE_FIREBASE_API_KEY`, `VITE_FIREBASE_AUTH_DOMAIN`, `VITE_FIREBASE_PROJECT_ID`, `VITE_FIREBASE_STORAGE_BUCKET`, `VITE_FIREBASE_MESSAGING_SENDER_ID`, `VITE_FIREBASE_APP_ID`.
