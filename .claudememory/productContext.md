# Product Context

- Users can browse questions in a public viewer, solve quizzes, upload questions through a passcode gate, and manage the question bank in an admin dashboard.
- The quiz flow persists learning-loop state so a user can continue from where they left off after refresh.
- Admin users can edit, delete, select, and export questions from Firestore.
- The app uses localized labels and dynamic document titles based on the current route and language.
